Header-only C++17 Library

BinarySearchTree.h

A clean, reusable Binary Search Tree for C++17 with smart-pointer ownership, STL-style iterators, comparator support, traversal helpers, and beginner-friendly documentation.

This library is a normal Binary Search Tree. It does not rebalance itself, so sorted input can degrade operations to linear time.

Quick Example

#include <iostream>
#include <bst/bst.h>

int main() {
    BinarySearchTree<int> tree = {8, 3, 10, 1, 6};
    tree.insert(14);
    tree.erase(3);

    for (int value : tree) {
        std::cout << value << ' ';
    }
    std::cout << '\n';
}

Highlights

  • Header-only C++17 design
  • `std::unique_ptr` node ownership
  • Comparator-aware ordering with `BinarySearchTree<T, Compare>`
  • STL-style `insert`, `emplace`, `find`, `contains`, and iterators
  • `lower_bound`, `upper_bound`, `min`, `max`, `height`, `to_vector`
  • In-order, pre-order, and post-order traversal helpers
