Header-only C++17 Library BinarySearchTree.h
Overview Getting Started API Reference Examples Performance

A modern header-only C++17 Binary Search Tree library with smart-pointer memory management, STL-style iterators, traversal helpers, comparator support, and a documentation set that can be browsed from GitHub or GitHub Pages.

This is a plain Binary Search Tree, not a self-balancing AVL or Red-Black tree. Sorted insertion order can degrade operations to O(N).

Quick Example

#include <iostream>
#include <bst/bst.h>

int main() {
    BinarySearchTree<int> tree = {8, 3, 10, 1, 6};
    tree.insert(14);
    tree.erase(3);

    for (int value : tree) {
        std::cout << value << ' ';
    }
    std::cout << '\n';
}

Public project highlights

  • Header-only C++17 layout with the public header at include/bst/bst.h
  • Comparator-aware BinarySearchTree<T, Compare>
  • STL-style insert, emplace, find, contains, and iterator APIs
  • Utility helpers including lower_bound, upper_bound, min, max, height, and to_vector
