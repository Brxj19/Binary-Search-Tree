`BinarySearchTree` is a small, reusable C++17 container for unique values stored in comparator order. The public API is intentionally beginner-friendly, but it still follows familiar STL-style patterns for insertion, lookup, erasure, and iteration.

This library is a normal Binary Search Tree. It does not rebalance itself, so sorted input can degrade search, insertion, and erase to linear time.

Library Overview

The implementation lives in include/bst/bst.h. Node ownership is managed with std::unique_ptr, values are unique, and iteration follows in-order traversal. Custom ordering is supported through the Compare template parameter.

Highlights

  • Header-only and C++17-compatible
  • Comparator-aware ordering with BinarySearchTree<T, Compare>
  • insert, emplace, erase, find, contains, and clear
  • Mutable and const bidirectional iterators
  • lower_bound, upper_bound, min, max, height, and to_vector
  • Traversal helpers for in-order, pre-order, and post-order visits
