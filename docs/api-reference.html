API Reference

The class template is declared as template <typename T, typename Compare = std::less<T>> class BinarySearchTree;. Values are unique, ordered by Compare, and exposed through bidirectional iterators.

The Markdown reference in docs/api-reference.md contains the full per-method write-up with prototypes, return values, notes, and cross-links. This HTML page is the published browsing view for GitHub Pages.

Constructors and Assignment

BinarySearchTree(const Compare& compare = Compare())

Constructs an empty tree using the supplied comparator.

BinarySearchTree<int> ascending;
BinarySearchTree<int, std::greater<int>> descending;

Complexity: constant.

BinarySearchTree(InputIt first, InputIt last, const Compare& compare = Compare())

Builds a tree by inserting a range of values. Duplicate values are ignored.

std::vector<int> values = {8, 3, 10, 1, 6};
BinarySearchTree<int> tree(values.begin(), values.end());

Complexity: average O(N log N), worst O(N^2).

BinarySearchTree(std::initializer_list<T> init, const Compare& compare = Compare())

Builds a tree from a braced list of values.

BinarySearchTree<int> tree = {8, 3, 10, 1, 6};

Complexity: average O(N log N), worst O(N^2).

BinarySearchTree(const BinarySearchTree& other) and operator=(const BinarySearchTree& other)

Create or assign a deep copy of another tree.

BinarySearchTree<int> source = {8, 3, 10};
BinarySearchTree<int> copy(source);
BinarySearchTree<int> assigned;
assigned = source;

Complexity: linear in the size of the copied tree.

BinarySearchTree(BinarySearchTree&& other) and operator=(BinarySearchTree&& other)

Move node ownership from another tree and leave the source empty.

BinarySearchTree<int> source = {8, 3, 10};
BinarySearchTree<int> moved(std::move(source));

Complexity: constant.

swap(BinarySearchTree& other)

Exchange the contents and comparator state of two trees without copying values.

BinarySearchTree<int> a = {1, 2};
BinarySearchTree<int> b = {8, 9};
a.swap(b);

Complexity: constant.

Capacity

size() const noexcept

Returns the current number of stored values.

BinarySearchTree<int> tree = {8, 3, 10};
auto count = tree.size();

Complexity: constant.

empty() const noexcept

Checks whether the tree contains any values.

BinarySearchTree<int> tree;
bool is_empty = tree.empty();

Complexity: constant.

Modifiers

clear() noexcept

Removes all nodes from the tree.

BinarySearchTree<int> tree = {8, 3, 10};
tree.clear();

Complexity: linear in size().

insert(const T& value) and insert(T&& value)

Insert a unique value and return {iterator, bool} indicating whether insertion occurred.

auto [it, inserted] = tree.insert(8);

Complexity: average O(log N), worst O(N).

insert(InputIt first, InputIt last)

Insert an entire input range.

std::vector<int> values = {10, 1, 6};
tree.insert(values.begin(), values.end());

Complexity: average O(M log(N + M)), worst O(M(N + M)).

emplace(Args&&... args)

Construct a value of type T from the forwarded arguments and insert it.

tree.emplace(7, "seven");

Complexity: average O(log N), worst O(N).

erase(const T& value)

Erase by key and return 1 on success or 0 if no matching value exists.

auto removed = tree.erase(3);

Complexity: average O(log N), worst O(N).

erase(const_iterator position)

Erase by iterator and return the next valid iterator.

auto it = tree.find(8);
auto next = tree.erase(it);

Complexity: average O(log N), worst O(N).

Lookup and Bounds

find(const T& value) and find(const T& value) const

Find an equivalent value and return an iterator or const iterator to it.

auto it = tree.find(6);
if (it != tree.end()) {
    std::cout << *it;
}

Complexity: average O(log N), worst O(N).

contains(const T& value) const

Convenience query that returns true when a matching value exists.

bool has_six = tree.contains(6);

Complexity: average O(log N), worst O(N).

lower_bound(const T& value) and const overload

Return the first element that is not ordered before the lookup value.

auto it = tree.lower_bound(5);

Complexity: average O(log N), worst O(N).

upper_bound(const T& value) and const overload

Return the first element ordered after the lookup value.

auto it = tree.upper_bound(6);

Complexity: average O(log N), worst O(N).

Iterators

begin(), begin() const, and cbegin()

Return an iterator to the smallest element in comparator order.

for (auto it = tree.begin(); it != tree.end(); ++it) {
    std::cout << *it << ' ';
}

Complexity: average O(log N), worst O(N).

end(), end() const, and cend()

Return the past-the-end iterator. On non-empty trees, --end() yields the maximum element.

auto it = tree.end();
if (!tree.empty()) {
    --it;
}

Complexity: constant.

Extremes and Helpers

min() and min() const

Return the smallest value. Throws std::out_of_range when the tree is empty.

int smallest = tree.min();

Complexity: average O(log N), worst O(N).

max() and max() const

Return the largest value. Throws std::out_of_range when the tree is empty.

int largest = tree.max();

Complexity: average O(log N), worst O(N).

height() const noexcept

Return the number of nodes on the longest root-to-leaf path.

auto h = tree.height();

Complexity: linear in size().

to_vector() const

Copy the tree contents into a vector using comparator order.

std::vector<int> values = tree.to_vector();

Complexity: linear in size().

is_valid_bst() const

Diagnostic helper that verifies the internal ordering still satisfies the BST property.

bool valid = tree.is_valid_bst();

Complexity: linear in size().

Traversal

in_order_traversal(UnaryFunction&& function) const

Visit values in comparator order.

tree.in_order_traversal([](const int value) {
    std::cout << value << ' ';
});

Complexity: linear in size().

pre_order_traversal(UnaryFunction&& function) const

Visit values in Root-Left-Right order.

tree.pre_order_traversal([](const int value) {
    std::cout << value << ' ';
});

Complexity: linear in size().

post_order_traversal(UnaryFunction&& function) const

Visit values in Left-Right-Root order.

tree.post_order_traversal([](const int value) {
    std::cout << value << ' ';
});

Complexity: linear in size().
